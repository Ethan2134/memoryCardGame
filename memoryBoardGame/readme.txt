Memory Card Guessing Game

This game was created as a class project using NetBeans IDE. This is a game where you guess two spots on the board, 
and try to get them to match. There is no guessing limit, so you can try as much as you like. To play, download the files and
open the command line. Navigate to the 'dist' folder and run the follwing command without quotation marks: "java -jar memoryBoardGame.jar". 
You can select the board size for a quicker or longer game.
